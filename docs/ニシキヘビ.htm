動的言語プラグイン > ニシキヘビ
ニシキヘビ

概要

IronPythonを操作します

必要プラグイン

動的言語プラグイン（PDlr.dll）

手順

手順名 助詞 結果
食わせる 【自分】に【ソースプログラム】を

ソースコードを実行して評価結果を返します

 

実行例

IronPythonで書かれたプログラムを実行して、戻り値を表示します。

「PDlr.dll」を利用する。
N=11
ニシキヘビに「
def fib(n):
if n <= 1:
return n
else:
return fib(n-1) + fib(n-2);
fib([N])
」を食わせる。
それを表示する。
